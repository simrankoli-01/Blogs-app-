# BlogSphere

A modern, full-featured blogging platform where every story deserves to be shared. Built with React and powered by Appwrite as the backend, it supports user authentication, rich text post creation, image uploads, and full CRUD functionality for posts.

## Features
- 🔐 User authentication (signup/login) via Appwrite
- ✍️ Rich text editing with TinyMCE
- 🖼️ Image upload and management via Appwrite Storage
- 📝 Create, edit, and delete blog posts
- 🎯 Author-only edit/delete permissions
- 📱 Fully responsive design
- ⚡ State management with Redux Toolkit

## Tech Stack
- **Frontend:** React, React Router, Tailwind CSS
- **Forms:** React Hook Form
- **Rich Text Editor:** TinyMCE
- **State Management:** Redux Toolkit
- **Backend/BaaS:** Appwrite (Database, Storage, Auth)
